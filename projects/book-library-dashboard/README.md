# 📚 Book Inventory Dashboard

A responsive book inventory dashboard built with HTML and CSS to display a collection of books in a clean and organized tabular layout. The interface categorizes books by reading status, highlights each category with distinct visual styles, and presents ratings using custom CSS components.

This project focuses on building structured tables, applying modern styling techniques, and creating an intuitive user interface with semantic HTML and responsive design principles.

---

## ✨ Features

- Responsive inventory table layout
- Book categorization by reading status
- Custom status badges with color coding
- CSS-based rating indicators
- Modern dark theme interface
- Responsive design for smaller screens
- Clean and semantic HTML structure

---

## 🛠️ Tech Stack

- HTML5
- CSS3
